## Features
- Classic snake gameplay
- AI mode (Autopilot)
- Simple and intuitive controls
- Multiple different foods
- Obstacles
- Local signup & login
- Saving highscores
- Local leaderboard
- Adjustin speed & game size


## Requirements
- Python 3.6+
- keyboard
- Pygame

## Usage
To start the game, simply run:
   ```sh
   snake
   ```

## Uninstallation
To uninstall the game, run:
   ```sh
   pip uninstall snake
   ```
